# Clario

Clario is a web-based AI tool that helps simplify complex text into easier-to-read content, especially for people with dyslexia.

## Problem

Many people, especially individuals with dyslexia, find it difficult to understand long and complex text. This can affect learning, access to information, and confidence in reading.

## Solution

Clario allows users to paste text and automatically converts it into a simpler and clearer version using AI.

## Features

* Text simplification
* Easy to use interface
* Fast processing

## Tech Stack

* HTML, CSS, JavaScript
* OpenAI API
* Vercel

## How to Run

1. Clone this repository
2. Open `index.html` in your browser
